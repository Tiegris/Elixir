# Elixir

## Install

1. Copy this folder to your home directory and rename it to .elixir

1. Build your docker image

    ```bash
    docker build -t elixir:tiegris .
    ```

1. Add the following line to your .bashrc

    ```bash
    source ~/.elixir/bash-extension.sh
    ```

## Commands

```bash
elixir-start
elixir
elixirc
iex
```
