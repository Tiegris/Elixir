# Elixir

## Install

1. Copy this folder to your home directory and rename it to .elixir

1. Add the following line to your .bashrc

    ```bash
    source ~/.elixir/bash-extension.sh
    ```

1. Run elixir-upgrade to build docker image and update extension.

    ```bash
    elixir-upgrade
    ```

## Commands

```bash
elixir-start
elixir
elixirc
iex
```
